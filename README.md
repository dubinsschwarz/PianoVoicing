This plugin automatically updates offset velocities of notes in piano compositions to improve the voicing.

The user can define default offset velocities for:
- notes at the top of a chord (RH + LH)
- notes not at the top of a chord (RH)
- notes at the bottom of a chord (LH)
- notes not at the top or bottom of a chord (LH)
- notes in secondary voices

Velocities are updated in <1s whenever the notes on the score are changed. This includes when existing chords are rearranged.

One limitation with this plugin is that it may overwrite manual velocity offsets if they coincide with any of the default values.

However, any offsets manually set to differ from the defaults are recognised as manual and protected.
